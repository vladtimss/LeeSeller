# Wildberries API

## Доступные фичи

### ping

Проверка подключения к WB API.

```bash
# Для всех магазинов
npm run executor -- wb ping

# Для конкретного магазина
npm run executor -- wb ping povar-na-rayone
npm run executor -- wb ping leeshop
```
